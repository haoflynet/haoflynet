并发框架

concurrent.futures

该库通过相同的API同时支持线程与协程，以POOL的方式进行并行任务的管理。其中Executors用于管理workers，而futures则用于表示一个异步计算的结果，当调用future时会被立即返回，但是不一定就是最终结果。

ThreadPoolExcutor改为ProcessPoolExecutor就是线程了，真方便。
