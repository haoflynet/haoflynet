## 环境搭建

命令行工具: `npm install -g react-native-cli`

测试安装: 

```she
react-native init testProject	# 新建项目目录
react-native init testProject --version 0.1.2	# 创建指定版本的项目
cd testProject
react-native run-ios	# 
react-native run-android
```
