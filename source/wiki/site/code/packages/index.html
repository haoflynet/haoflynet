＃ python包／jQuery包收集与整理

Python

如果要使用pip安装最新版本可以在后面加上版本号

beautifulsoup4：XML/HTML解析组件

coverage：代码覆盖率检测工具

difflib：Python自带模块，比较文本之间的差异，且支持输出可读性强的HTML文档，

django.core.paginator：这是Django自带的分页工具，非常实用

django.contrib.syndication.views：Django自带的输出feed的工具

django-avatar：Django头像插件

django-debug-toolbar：Django的调试工具，应该非常好用，未使用过

django-haystack：Django的全文搜索功能

django-redis：在Django中使用Redis必备。需要注意的是，它对value做了序列化，而且在key前面加入了版本号，类似_:1:key，而且，默认生存时间是300秒，需要加入参数_cache.set(“key”, “value”, timeout=None)。Redis密码的格式应该是 “LOCATION”: “redis://:密码内容@104.236.170.169:6379/1″,真的服了官网那不明不白的表述了

django-rest-framework：Django的Restful框架

django-social-auth：Django社会化认证工具

django-socketio：Django的WebSockets ，好爽

dnspython：DNS工具包，支持几乎所有的记录类型，可用于查询、传输并动态更新ZONE信息，同时支持TSIG(事务签名)验证消息和EDNS0(扩展DN S)。我们可以利用其查询功能来实现DNS服务监控以及解析结果的校验，可以代替nslookup及dig等工具。

django-wysiwyg：Django使用wysiwyg作为富文本编辑器

grequests：异步请求

IPy：IP地址处理模块，可用于计算大量的IP地址，包括IPv4、IPv6网段、网络掩码、广播地址、子网数、IP类型等。参考文章

jieba(官方文档)：结巴中文分词，未使用过

matplotlib：像Matlab那样使用图表的方式展示数据，未使用过

MkDocs(中文文档)：项目文档工具，以markdown的方式攥写spinx烈性的文档
MoviePy：Python处理视频文件

mysqlclient：Python3连接MySQL数据库，相比于官方的python connector它更新更快，而且安全，所以我选择它，在安装的时候需要先安装依赖：sudo apt-get install python-dev libmysqlclient-dev，Python3要加3哟，windows下可以直接安装，如果是OS X，那么可能是没有将mysql添加到环境变量，在_.profile_做如下修改

PATH="/Library/Frameworks/Python.framework/Versions/3.5/bin:/usr/local/bin/pyt hon3:/usr/local/mysql/bin:${PATH}"

export PATH

DYLD_LIBRARY_PATH="/usr/local/mysql/lib/${DYLD_LIBRARY_PATH}"

export DYLD_LIBRARY_PATH

---|---

Pandas：大数据处理工具，可将大文件分段读取

paramiko(官方文档)：基于Python2/3实现的SSH2的库，支持认证及密钥方式，可以实现远程命令执行、文件传输、中间SSH代理等功能。windows安装的时候会有依赖问题，可见这个issue

pep8：PEP8规范检测工具，使用时直接pep8 ./

pexpect：相当于Linux下得expect，可以实现对ssh、ftp、passwd、telnet等命令进行自动交互。比如可以模拟一个FTP登录时的所有 交互，包括输入主机地址、用户名、密码、上传文件等等。

Pillow：Python图像处理库，与PyLab互斥，只能安装一个哟

progressBar2：在终端显示进度条

psutil：跨平台的获取系统运行的进程和系统利用率(包括CPU、内存、磁盘、 网络等)信息的库，主要用于系统监控，分析和限制系统资源及进程的管理。实现了一些命令行的工具(如：ps、top、lsof、netstat、ifconfig、w ho、df、kill、free、nice、ionice、iostat、iotop、uptime、pidof、tty、taskset、pmap等)

PyBrain：Python神经网络库

pyClamad：直接使用ClamAV病毒扫描守护进程clamd，来实现一个高效的病毒检测功能(ClamAV是一款免费而且开源的防毒软件，软件与病毒库的更新 皆由社区免费发布)

pycurl：支持的操作协议有FTP、HTTP、HTTPS、TELNET等

pyspider：有图形界面的爬虫程序

python-markdown2：Python解析markdown文件

python-nmap：使用Python实现的端口扫描工具

randomavatar：根据字符串生成随机头像 ，在Python3下有点问题

random-avatar：直接生成指定大小的随机头像，是按照你的IP来计算的

requests(官方文档)：比SSL和HttpResponse更加高级，更方便，一句话就可以搞定人家几十句的功能，非常方便 |

如果出现以下错误：

/usr/local/lib/python2.7/dist- packages/requests/packages/urllib3/util/ssl_.py:90: InsecurePlatformWarning: A true SSLContext object is not available. This prevents urllib3 from configuring SSL appropriately and may cause certain SSL connections to fail. For more information, see https://urllib3.readthedocs.org/en/latest/security.h tml#insecureplatformwarning.

InsecurePlatformWarning

那么这样操作：

pip uninstall requests

pip \--no-cache-dir install requests[security]

如果出现如下错误：

[APP SUBMITTED]: '_socketobject' object has no attribute 'set_tlsext_host_name'

那么这样操作：

apt-get purge python-openssl

apt-get install libffi-dev python-pip

pip2.7 install -U pyopenssl

SciPy：Python科学计算库

scrapy-splash：Scrapy对于Splash的封装

splash：用于获取经过Javascript处理后的html页面，十分有用，只需要将请求的url改为http://localhost:8050/render.html?url=' + url，即可向Splash服务器发送，并获取到js渲染过后的html页面

suds：Python仅作为客户端调用webservice

Tornado-MySQL：MySQL对于Tornado的异步实现，支持线程池直接操作

xpinyin：汉字拼音

laravel-5-markdown-editor：Laravel5 Markdown编辑器

PHP Debug Bar：方便调试，可以直接在浏览器里面看到变量信息，而不用var_dump()了。

clipboard.js：纯HTML5实现的复制到粘贴板的插件

fingerprintjs: 浏览器唯一性解决方案
