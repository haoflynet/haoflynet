PHP

基本语法

数组

array_key_exists("key",$a)  # 查看key是否存在于某个数组
count()函数：输出数组或字符串的长度
sort()：排序，可以给数组排序
array_sum($arr): 计算数组中所有值的和
array_chunk($array, $size): 将数组按size大小分为多个数组
array_push($source, "red", "gree")  # 给数组添加元素
foreach($array as $value): 数组遍历
foreach($array as $key => $value): 数组(字典)遍历

字符串

json_decode(string, $assoc=false)   # 将字符串转换为json对象, $assoc=true的时候返回array而非object
strlen()：求字符串长度
mb_strlen($str, 'utf-8')：求中文字符串长度
strpos('abc', 'a'): 在字符串中查找第一次出现位置，没找到返回false
$a . $b . 'abc':字符串连接直接用点号
explode(',', $str)  # 字符串分割

数字

ceil()函数：向上取整
rand(min, max)：产生随机数
intval($val): 字符串转整数

WEB程序

$_SERVER['REQUEST_METHOD']  # 返回数据提交的方式，GET、POST等
parse_str(file_get_contents("php://input"),$post_vars); # 获取PUT数据

帮助函数

  • @操作符: 错误控制运算符，写在一行的前面，可以控制改行不输出warning信息或错误信息

  • var_dump(变量名)：打印变量，这个函数还会打印变量的类型可以把一个变量的各个部分全部信息输出，包括每个部分的数据类型和长度等信息，但是默认情况下，输出有限制，如果层数深了或者数据长了可能会表示成省略号，可以在C:\wamp\bin\apache\apache2.4.9\bin\php.ini里面修改xdebug节点，添加如下内容

    xdebug.var_display_max_children=128
xdebug.var_display_max_data=512
xdebug.var_display_max_depth=5
    • file_get_contents：获取文件或http内容，如果要从http获得json数据可以直接使用它
    • isset()：查看某个变量是否已经被定义，未赋值或赋NULL都会返回false

  • Unix()时间戳，是10位整型

    time()：获取当前的时间戳

    $beginToday=mktime(0,0,0,date('m'),date('d'),date('Y'))：获取今天开始时的时间戳 $endToday=mktime(0,0,0,date('m'),date('d')+1,date('Y'))-1：获取今天结束时的时间戳 $beginYesterday=mktime(0,0,0,date('m'),date('d')-1,date('Y'))：获取昨天开始时的时间戳 $endYesterday=mktime(0,0,0,date('m'),date('d'),date('Y'))-1：获取昨天结束时的时间戳 $beginLastweek=mktime(0,0,0,date('m'),date('d')-date('w')+1-7,date('Y'))：获取上周开始时的时间戳 $endLastweek=mktime(23,59,59,date('m'),date('d')-date('w')+7-7,date('Y'))：获取上周结束时的时间戳 $beginThismonth=mktime(0,0,0,date('m'),1,date('Y'))：获取本月开始时的时间戳 $endThismonth=mktime(23,59,59,date('m'),date('t'),date('Y'))：获取本月结束时的时间戳

  • 单独获取当前的年、月、日、时、分、秒等

    date('Y-m-d H:i:s'); // 如果要单独获取或者修改格式，那么直接按照里面的格式修改即可

  • 将某个时间戳按指定格式输出

    date('Y-m-d H:i', time())

    字符串分隔： mb_substr($str, $start, $length, 'utf-8')：分隔中文字符串，如果是要获得右边的，$length填上null $hello = explode(',', $str)：将字符串$str按逗号分隔开来，并以数组方式返回

    转换： int ip2long(string $ip_address)：IP转换成整数值 string long2ip(string $proper_address)：整数值转换成IP

    1. php文件中添加中文支持，只需在开始的地方添加

      @header('Content-type: text/html;charset=UTF-8');

    2. PHP多行输出，其中最后一个EOF必须写在一行的开头，且里面如果要用变量这样用{ $php_var }

      echo <<<EOF 内容 EOF;

    3. print_r：打印关于变量的易于理解的信息。如果给出的是 string、integer 或 float，将打印变量值本身。如果给出的是 array，将会按照一定格式显示键和元素。这点在调试的时候很有用。

    4. 使用内嵌的HTML代码比PHP的echo语句效率更高 因为每一个PHP页面只调用PHP解释器来解释所有的PHP代码，多用<?php ?>可以减少对echo语句的解析

    5. 类的方法尽量写成static，速度比public快
